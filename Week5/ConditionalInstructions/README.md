# Challenges

- I learned that I can’t compare two memory variables directly.
I had to use a register as a middleman to hold the largest value during comparisons.
- It was tricky setting up the labels so the code only updates the register when a larger number is found, otherwise jumping past the update.
- Making sure the even sequence stopped exactly at 20 required using the right condition after the comparison.
